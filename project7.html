INFOS PROJET

Année: 2023

Catégorie: Architecture open source

Sujet: Creation de librairie

Logiciel: Visual Studio Code

Langage: JavaScript

Ce projet scolaire avait pour objectif de nous initier a la creation de librairie open source. L'idée était des fonctionnalites que les autres devellopeur pourrait utuliser dans leur code. Nous nous sommes concentré sur la traduction de mot, correction de mot et dection d'une langue
