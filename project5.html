INFOS PROJET

Année: 2022

Catégorie: Développement web

Sujet: Site internet

Logiciel: Visual Studio Code

Langage: HTML SCSS

VISITER LE SITE

Ce projet est un projet scolaire. Après une semaine qui m'a permis de découvrir le SCSS, j'ai dû réaliser un site web classique sur un monument ou un lieu situé à Paris ou ses alentours. Le lieu que j'ai choisi de présenter est le château de Vincennes. La contrainte était d'utiliser du SCSS au lieu du CSS pour la mise en page.
