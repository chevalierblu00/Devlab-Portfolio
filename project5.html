INFOS PROJET

Année: 2023

Catégorie: Développement web

Sujet: système de contact téléphonique

Logiciel: Visual studio code

language: vue js

VISITER LE SITE

Ce projet scolaire avait pour objectif de nous initier au framework Vue js. L'idée était de reproduire les fonctionnalités de base de la fonction "contact" sur les téléphones. Nous devions permettre la composition d'un numéro de téléphone avec une fonction de suppression. Une fois composé, il s'afficherait dans l'historique et nous pourrions en rajouter dans l'onglet "contacts".
