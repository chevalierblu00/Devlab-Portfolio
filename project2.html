INFOS PROJET

Année: 2023

Catégorie: Développement web

Sujet: API

Logiciel: Visual Studio Code

Langage: Vue.js

VISITER LE SITE

Ce projet scolaire avait pour objectif de nous initier aux API. L'idée était de reproduire un Pokédex en affichant uniquement les dix premiers Pokémons de l'API Pokémon. De plus, nous devions être en mesure de créer des équipes et d'ajouter des mini-jeux.
